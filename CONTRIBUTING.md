# Contributing to Vuact

## Quick Start

### Prerequisites

- Node.js 20+
- pnpm 10+

### Installation

1. **Clone the repository**

``` bash
git clone https://github.com/yinhangfeng/vuact.git
```

2. **Install project dependencies**

``` bash
pnpm i
```

3. **Run examples**

``` bash
pnpm examples
```

4. **Build the project**

``` bash
pnpm build
```
